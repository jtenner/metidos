# Tactile & Haptic Accessibility

Use this skill when reviewing products that use vibration or tactile feedback. Focus on whether haptics help users without becoming the only or dominant way important information is communicated.

## Review Haptic Use

- Identify confirmations, alerts, navigation cues, or patterns conveyed through vibration
- Check whether users can perceive, distinguish, or control the haptic signal
- Consider users with reduced tactile sensitivity, prosthetics, gloves, mounting, or assistive devices
- Review whether critical meaning is lost when haptics are unavailable

## Improve Inclusivity

- Pair haptics with visible, textual, or audible alternatives as appropriate
- Make vibration patterns simple, purposeful, and customizable where possible
- Avoid excessive or startling feedback
- Ensure important system state can be understood without tactile cues

## Output

- Highlight over-reliance on tactile signaling
- Recommend more inclusive feedback combinations
- Prioritize issues where haptic-only communication blocks understanding
