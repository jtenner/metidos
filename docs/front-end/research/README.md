# Research Notes

Use this folder for dated, source-backed investigations.

Recommended format for each research note:

- Title with a specific topic
- Date of the research pass
- Short summary of what was learned
- Links or citations to the web sources used
- Actionable implications for the front-end docs tree

When a research note becomes stable guidance, move the distilled material into `../principles/` or `../patterns/` and leave behind only the source-oriented note.
