# Cognitive Load Assessment

Use this skill when assessing or designing interfaces for users with varying cognitive capacities. Focus on identifying unnecessary complexity and reducing effort required to understand, navigate, and complete tasks.

## Assess

- Count simultaneous decisions, options, and instructions shown at once
- Identify multi-step processes requiring users to remember previous info
- Flag jargon, dense paragraphs, or unnecessary visual noise
- Prioritize simplification where users may be stressed, distracted, or unfamiliar

## Improve

- Reduce choices at each step; present only what matters now
- Chunk information into smaller sections with clear headings
- Use consistent patterns, predictable interactions, and plain language
- Support recognition over recall with visible labels, examples, and progress cues

## Output

- Call out likely overload points
- Suggest specific simplifications
- Recommend changes that improve comprehension without reducing capability
