# Mobile Form Input

Use this skill when reviewing mobile forms. Focus on reducing effort, improving clarity, and supporting error-free completion on small screens and touch keyboards.

## Evaluate Entry Friction

- Check field labels, hints, grouping, and order on small screens
- Review keyboard types, autocomplete, input masks, and platform autofill behavior
- Identify fields that demand unnecessary typing or precision
- Consider interruptions, context switching, and fatigue during entry

## Improve Completion

- Use appropriate input modes and minimize manual typing
- Keep labels visible and instructions concise
- Place validation and error recovery close to the relevant field
- Support paste, password managers, and accessible date/time selection

## Output

- Highlight mobile-specific form barriers
- Recommend changes that improve speed, accuracy, and recovery
- Prioritize improvements for critical conversion or task flows
