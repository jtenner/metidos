# Metidos Installation

The canonical installation guide now lives at [`docs/installation.md`](docs/installation.md).

Use:

- [`docs/getting-started.md`](docs/getting-started.md) for the shortest clean-clone path to first project and first Thread.
- [`docs/installation.md`](docs/installation.md) for local development, local production, containers, first-run auth, provider setup, backups, restore, reverse proxy, and Tailscale-style access.
- [`.pi/skills/metidos-installation/SKILL.md`](.pi/skills/metidos-installation/SKILL.md) for the interactive installer/operator workflow.

The installer workflow is intentionally plan-first: it should ask questions, build a plan, export `metidos-config.md` without raw secrets, and apply host changes only after explicit approval.
