# Target Size & Spacing

Use this skill when assessing touch interfaces for motor accessibility and error prevention. Focus on whether users can reliably activate controls without precision or accidental input problems.

## Evaluate Touch Targets

- Check size and spacing of buttons, links, icons, toggles, and custom controls
- Identify dense clusters, edge-aligned controls, or small hit areas
- Consider thumb reach, one-handed use, tremors, limited dexterity, and low vision
- Review how target size changes across breakpoints or embedded components

## Improve Safety & Comfort

- Increase interactive area without relying on precise visual targets
- Add spacing between adjacent actions, especially destructive ones
- Avoid placing important controls too close to screen edges or competing gestures
- Support clear feedback and recovery from accidental activation

## Output

- Flag target size and spacing risks in key flows
- Recommend practical adjustments based on context and priority
- Emphasize changes that reduce accidental taps and physical effort
