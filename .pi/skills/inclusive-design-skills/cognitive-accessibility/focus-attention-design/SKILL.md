# Focus & Attention Design

Use this skill when evaluating whether an interface supports concentration and task persistence. Focus on reducing interruptions, visual competition, and attention demands that make completion harder.

## Assess Attention Demands

- Identify auto-updating content, popups, banners, or secondary actions competing for focus
- Review visual hierarchy and whether the next step is obvious
- Check for time pressure, multitasking assumptions, or dense pages with many competing elements
- Consider users with ADHD, fatigue, anxiety, or stress

## Improve Focus Support

- Emphasize the primary task and reduce competing calls to action
- Break long tasks into shorter stages with clear progress cues
- Minimize unnecessary motion and interruptions
- Provide reminders, save/resume, and visible next steps where helpful

## Output

- Point out where attention is likely to fragment
- Suggest simplifications that help users stay oriented and complete tasks
- Prioritize changes that reduce abandonment and confusion
