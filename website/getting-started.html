Getting started

Get Metidos installed with the /install skill.

Keep setup simple: install the requirements, run your agent harness, then let the /install skill guide the rest.

Full install guide Docs

Install requirements

Install these first.

You only need the basics before the guided installer can take over.

  • Docker or Podman.
  • Git.
  • An agent harness that can run the /install skill.
  • The latest Bun version.

Install instructions

Run the guided setup.

The installer will ask questions, check requirements, and tell you what to do next.

  1. 1. Run your agent harness.
  2. 2. Run /install.
  3. 3. Follow the agent instructions.
