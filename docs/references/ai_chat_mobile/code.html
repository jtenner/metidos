How can I optimize this React component to prevent unnecessary re-renders of heavy list items? I'm currently using regular props but it feels sluggish. I've heard about memoization.
hub Intelligence

To optimize heavy list items in React, you should implement selective memoization. This ensures that only items whose data has actually changed will re-render.

ListItem.tsx content_copy
import { memo } from 'react'; const ListItem = memo(({ data, onSelect }) => { console.log('Rendering:', data.id); return ( <div onClick={() => onSelect(data.id)}> {data.label} </div> ); }, (prev, next) => { return prev.data.id === next.data.id && prev.data.updatedAt === next.data.updatedAt; });

By providing a custom comparison function as the second argument to memo(), we can ignore functional prop changes (like onSelect) if they aren't stable references.
