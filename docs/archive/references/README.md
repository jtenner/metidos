# References

This folder stores UI reference snapshots used for visual alignment and feature validation.

Each subfolder contains:
- `code.html`: captured DOM/state artifact or generated markup sample
- `screen.png`: corresponding screenshot

## Reference sets

- `ai_chat_mobile/`
  - Mobile AI chat layout reference.
- `codex_view/`
  - Codex-oriented workspace layout and interaction reference.
- `compact_diff_view_desktop/`
  - Desktop diff-view compact layout reference.
- `compact_diff_view_mobile/`
  - Mobile compact diff-view reference.
- `file_editor_mobile/`
  - Mobile file editor view reference.
- `file_tree_view/`
  - File tree navigation/reference state.
- `task_list_mobile/`
  - Mobile task list UI reference.
- `updated_tasks_view/`
  - Updated task list/workspace behavior reference.
